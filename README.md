<h1>sGym - Fitness Management System</h1>
<p>SGym is a comprehensive mobile application developed using the Flutter framework, designed to facilitate the management of fitness centers, scheduling, and trainer-client interactions. The project is engineered with a focus on scalability, testability, and maintainability, adhering to the principles of Clean Architecture and SOLID design patterns. It leverages Firebase for backend services to ensure real-time data synchronization and secure authentication.</p>
